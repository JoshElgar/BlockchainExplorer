API Documentation

This API allows you to query various information about the network and it's state, as well as information about individual blocks and transactions.

Base URL

The API is served over HTTP, and all accessible URLs are accessed with the following base:

http://localhost:8080/api

For example: 

http://localhost:8080/api/search/b081a9e6ff7abe3947c29bf4d5fd2fa6407d9457cc59b9d71321f4183ec61301


Endpoints

Endpoints are the unique URLs that represent access to a specific set of data (resource). The following endpoints are provided:

Search
/api/search/{value}

This returns a Map<String, String> where the key represents the type of value being returned (either "tx", "block" or "none") and the value represents either a txid or a block hash, or is simply "none" if no matching Tx/Block could be found. The {value} can be either a block hash, a txid or a tx hash.

{
    "tx": "12b2af9a8cf12c9650cef12f73394afde24a039d0574af4a92c85212405eb9b2"
}
